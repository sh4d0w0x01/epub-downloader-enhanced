## 🚀 Quick Start

### Installation

1. **Clone this repository**
   ```bash
   git clone https://github.com/security-log/epub-downloader.git
   cd epub-downloader
   ```

2. **Open Firefox and load the extension**
   - Type `about:debugging` in address bar
   - Click "This Firefox" → "Load Temporary Add-on"
   - Select `extension/manifest.json`

3. **Log in to O'Reilly**
   - Go to [learning.oreilly.com](https://learning.oreilly.com)
   - Log in with your account

4. **Open a book and download**
   - Open any book you have access to
   - Click extension icon in toolbar
   - Click "Download EPUB"
   - EPUB saves to your Downloads folder ✅
